Este repositorio contiene el proyecto final de desarrollo web realizado por **Claudia González Aparicio**. La web ofrece servicios de programación web con un enfoque moderno y funcional, cumpliendo con todos los requisitos estructurales solicitados.

## 🚀 Características
- **Página de Inicio**: Carga dinámica de noticias mediante `fetch` desde un archivo JSON externo.
- **Galería**: Visualización de proyectos mediante una cuadrícula (grid) dinámica generada con JS.
- **Presupuesto**: Calculadora interactiva con validaciones y cálculo de costes en tiempo real.
- **Contacto**: Información de contacto profesional y mapa de ubicación (La Lomba, Cantabria).

## 📂 Estructura del Proyecto
Siguiendo las directrices del equipo docente, el proyecto se organiza así:
- `/assets/icons`: Logos e iconos de redes sociales.
- `/css`: Hoja de estilos unificada (`style.css`).
- `/data`: Archivos de datos externos (`noticias.json`).
- `/js`: Lógica de la aplicación (`main.js`, `galeria.js`, `presupuesto.js`).
- `/views`: Archivos HTML secundarios (`galeria.html`, `presupuesto.html`, `contacto.html`).
- `index.html`: Punto de entrada principal en la raíz.

## 🛠️ Tecnologías utilizadas
- HTML5 / CSS3 (Flexbox y Grid)
- JavaScript (Fetch API / Manipulación del DOM)
